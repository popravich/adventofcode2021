# adventofcode2021

Advent Of Code 2021

## CLI variant of code.

```
    cargo run --bin X
```

where `X` is the day number.

## WebAssembly variant of code.

### How to run

```
    # install and serve js (in other terminal)
    cd www
    npm install
    npm run start

    # build wasm code
    wasm-pack build
```
